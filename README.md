# Kubernetes
Этот проект — учебная площадка для знакомства с основами Kubernetes. Здесь отрабатываю создание манифестов, развертывание подов через Deployments и настройку сетевого доступа с помощью Services. Основная цель: на практике разобраться в архитектуре K8s и научиться управлять контейнеризированными приложениями.
